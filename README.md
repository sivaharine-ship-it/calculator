# Smart UI Calculator

A sleek, modern, and responsive web-based calculator designed with a beautiful dark-mode interface. Built purely with HTML, CSS, and Vanilla JavaScript.

![Calculator Preview](https://the-calculatorr.netlify.app/)

## 🚀 Features

- **Modern Dark UI:** Clean aesthetic utilizing a refined dark color palette with vivid accent highlights.
- **Responsive Layout:** Perfectly centered and adaptive using CSS Flexbox and Grid.
- **Dynamic Interactions:** Smooth hover effects and micro-interactions when keys are pressed.
- **Full Calculation Suite:**
  - Standard operations: `+`, `-`, `*`, `/`
  - Percentage computation (`%`)
  - Sign toggling (`+/-`)
  - Backspace deletion (`DEL`) and full clear (`C`)
- **Smart Decimal Truncation:** Automatically formats long floating-point results up to 4 decimal places to prevent overflow.
- **Graceful Error Handling:** Displays a timed "Error" prompt during invalid mathematical evaluations rather than breaking the application.

## 🛠️ Technologies Used

- **HTML5:** Structured semantic markup.
- **CSS3:** Custom styling utilizing CSS Grid, Flexbox, transitions, and transform animations.
- **JavaScript (ES6):** Clean vanilla DOM manipulation and calculation logic.
